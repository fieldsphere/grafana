# Amethyst Theme Artifacts

This feature adds the new **Amethyst** theme to Grafana's theme switcher (behind `grafanaconThemes`).

## Captured artifacts

- Screenshot: `.cursor/docs/artifacts/amethyst-theme-switcher.png`
- Video walkthrough: `.cursor/docs/artifacts/amethyst-theme-walkthrough.webm`

## Capture method

- Environment: local Grafana dev instance with `GF_FEATURE_TOGGLES_ENABLE=grafanaconThemes`
- Flow: login as `admin`, open **Profile** preferences, select **Amethyst** in Interface theme, save
- Script used: `.cursor/docs/artifacts/capture-amethyst-artifacts.mjs`
